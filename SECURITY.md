# Security Policy

## Supported Versions

Minimap is currently pre-1.0. Security fixes will target the latest released version.

## Reporting a Vulnerability

Please report security issues privately to the maintainers before opening a public issue.

If a private security advisory is available on GitHub, use that first. Otherwise, contact Forjd through the repository owner channels and include:

- A concise description of the issue.
- Steps to reproduce.
- Impact and affected versions, if known.
- Any suggested mitigation.

Minimap is designed to be local-only and does not execute project scripts during scan or generation. Reports involving unsafe file writes, command execution, or data exposure are especially relevant.
